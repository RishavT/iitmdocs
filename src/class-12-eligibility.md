# Class 12 Eligibility and Requirements

## Basic Eligibility
- Must have passed Class 12 or equivalent examination
- Must have studied Mathematics and English in Class 10
- No age restrictions

## Class 11 Students
- Can attempt qualifier exam after completing Class 11 finals
- Any group/stream/board accepted
- Can join Foundation level only AFTER passing Class 12

## Before Completing Class 12
You CAN:
- Register for qualifier
- Complete qualifier coursework
- Attempt qualifier exam
- Clear the qualifier exam

You CANNOT:
- Register for Foundation courses without Class 12

## Qualifier Validity for Class 12 Students
- For those who cleared qualifier before passing Class 12
- Validity starts from when they pass Class 12
- Valid for 3 terms from Class 12 completion

## Without Class 12 Completion
If you did not complete 10th and 12th grade, you are NOT eligible for the qualifier exam. Class 12 or equivalent is mandatory.

## Accepted Class 12 Equivalents
Diplomas and other qualifications may be accepted. Check:
https://study.iitm.ac.in/ds/accepted-class-12-equivalents.html

## Failed Class 12 Earlier
If you failed Class 12 earlier but passed it later, you ARE eligible to apply. You can write the qualifier exam.

## Age 40 or Above
No age restrictions. If you have passed Class 12 or equivalent, you can apply regardless of age.

## Commerce/Arts Students
Yes, non-science and commerce stream students can apply for BS in Data Science, provided they studied Mathematics and English in Class 10.
