# Exam Cities for IIT Madras BS Program

Quizzes and exams for the IIT Madras BS Program are conducted in person at exam centres across India and abroad.

## Selecting Exam Cities

- Students pick 2 cities among the choices provided for their quizzes and exams for each term
- Students have the freedom to choose a different set of cities than the ones they opted for in previous terms
- Students can choose a different set of cities for every quiz/final exam while registering for courses
- Booking will be done with the Exam vendor for slots in these selected cities
