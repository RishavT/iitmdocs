Answer: OPPE (Online Proctored Programming Exam) is conducted only for programming-based courses, where students need to write and execute code. Not all courses have OPPE.
Only courses that require hands-on coding evaluation use OPPE.

Typically, OPPE is applicable to courses such as: Programming / Python-related courses ,Data structures or applied programming courses (mainly in Foundation and Diploma levels of BS Data Science)
