# Assignment Grading and Category-wise Cutoffs

## Keywords
qualifier assignment, qualifier assignment requirements, assignment grading, qualifier exam eligibility, category-wise cutoff, minimum assignment score, SC ST OBC EWS cutoff, qualifier exam requirements

## Assignment Grading
- Each assignment will be graded out of 100
- Any assignment that is not attempted will be marked as 0

## Minimum Average Assignment Score by Category

Only those who get the minimum required average assignment scores in all four courses will be allowed to appear for the Qualifier Exam.

| Category | Minimum Average Assignment Score Required in Each Course |
|----------|----------------------------------------------------------|
| General Learner | 40% |
| SC / ST / PwD with 40% disability | 30% |
| PwD with 40% disability & SC / ST | 30% |
| OBC-NCL / EWS | 35% |

## Important Notes
- Relaxations in pass criteria indicated for various categories of learners are applicable ONLY for the qualifier process
- There will be no relaxations in terms of grades or pass criteria once registered into the program
- Hall tickets will be released only for candidates eligible to take the in-person proctored Qualifier Exam
